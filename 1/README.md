# HelloWorld Java 程序

这是一个简单的 Java 程序，打印 `Hello, World!`。

要求：已安装 JDK，并且 `javac` 与 `java` 在系统 PATH 中可用。

在 PowerShell 中的编译与运行步骤：

```powershell
# 进入源码所在目录
cd d:\作业\1

# 编译
javac HelloWorld.java

# 运行
java HelloWorld
```

注意：如果 `d:\作业\1` 路径含空格或特殊字符，请使用引号：

```powershell
cd "d:\作业\1"
javac HelloWorld.java
java HelloWorld
```
